# 🎭 Learning Playwright

A hands-on project for learning **Playwright** — Microsoft's end-to-end testing framework for modern web applications.

## 📋 About

This repository contains automation scripts built with [Playwright](https://playwright.dev/) to explore and practice browser automation, UI testing, and web scraping concepts.

## 🛠️ Tech Stack

- **Runtime:** Node.js
- **Framework:** Playwright
- **Language:** JavaScript

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm (comes with Node.js)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/amrutapatil27/LearningPlaywright.git
   cd LearningPlaywright
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Install Playwright browsers:**
   ```bash
   npx playwright install
   ```

### Verify Setup

Run the system check script to verify your environment:

```bash
node check_Syastem_Setup.js
```
